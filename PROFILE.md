# Thordata

Proxy network and web-data-scraping infrastructure providing residential, ISP, datacenter, and mobile proxies plus scraping APIs (Web Scraper API, Web Unlocker, SERP API) and datasets. Exposes a token-authenticated REST Public API for account and proxy-fleet management.

- **Provider:** https://www.thordata.com
- **Documentation:** https://doc.thordata.com/interface-documentation
- **Base URL:** https://openapi.thordata.com/api/

## APIs

- **Thordata Public API** — Token-authenticated REST API for account and proxy-fleet management (traffic/wallet balance, locations, proxy users, whitelist IPs, ISP/datacenter proxy lists), plus proxy/scraper data products documented alongside. (`https://openapi.thordata.com/api/`)

> REST Public API confirmed at https://openapi.thordata.com/api/ with token auth (token obtained from dashboard.thordata.com/account-settings). A second llms.txt at www.thordata.com/llms.txt is only a crawler-policy file. No public machine-readable contract (OpenAPI/AsyncAPI/GraphQL/Postman) was found. An MCP server exists (github.com/Thordata/thordata-mcp-server) but is local/self-hosted (installable, runs client-side) with NO hosted remote MCP endpoint, so it is not cataloged as an MCP property. Adjacent AI tooling includes LangChain tools, a RAG pipeline repo, and SDKs (Python flagship, Node/TS, Go, Java; thordata-sdk on PyPI).
